# Pixel Elevate: Isometric Healing Shooter

A pixel-art isometric game built with Pygame, where you **heal allies** and **elevate souls** instead of shooting and dying.

## Game Concept
- **Genre**: Isometric "healer-shooter"
- **Core Mechanic**: Healing replaces shooting; "elevating" (spiritual ascension) replaces enemy death.
- **Art Style**: Retro pixel art, mystical vibes.

## Classes
1. **Monk** — Close-range assassin healer (fast melee purification)
2. **Priest** — AoE blessing master
3. **Exorcist** — Ranged projectile healer

## How to Run
```bash
pip install pygame
python main.py
